# **App Name**: IPL Auction Clash

## Core Features:

- Room System: Allow users to create or join a room using a unique Room ID for private auctions.
- IPL Team Selection: Enable users to select an IPL team, making it unavailable to others once chosen.
- Real-Time Player Auction: Implement a live bidding system with a countdown timer and real-time bid updates visible to all users.
- Playing XI Reference: Provide a Playing XI layout for each team, used only as a visual reference without locking or restrictions.
- Trophy Winner Analysis: Analyze teams post-auction based on recent player form and basic stats, outputting a ranking of potential winners; the system will act as a tool, incorporating or ignoring such information in its assessment.
- Ranking Display: Display the final rankings of all participating teams after the analysis.
- User Authentication: Use Firebase Anonymous Login to authenticate users.

## Style Guidelines:

- Primary color: Vibrant orange (#FF9933), evoking the energy and excitement of IPL matches.
- Background color: Light, desaturated orange (#FFE5CC), creating a warm and inviting atmosphere.
- Accent color: Analogous yellow-orange (#FFB84D), to highlight interactive elements and call-to-actions.
- Headline font: 'Poppins', sans-serif, for a contemporary, fashionable, precise look. Body text: 'PT Sans', sans-serif, blending a modern look and a touch of warmth or personality.
- Use cricket-themed icons, like bats, balls, and trophies.
- Design a responsive layout that adapts to different screen sizes.
- Use subtle animations when players are bid on and assigned to teams.
